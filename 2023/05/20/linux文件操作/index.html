Open函数

头文件：
#include<sys/types.h>
#include<sys/stat.h>
#include<fcntl.h>
参数：

  1. -pathname:
    oen函数中的第一参数pathname代表想要打开文件的文件的路径名
  2. -flags:
    对文件的操作权限设置和其他设置，必须指定以下三个常量的一种，且只允许指定一个
  • o_RDONLY读
  • R_WRONLY写
  • O_RDWR读+写
    以下的常量是选用的，这些选项是用来和上面的必选项进行按位或起来作为flags参数。
  • O_APPEND 表示追加，如果原来文件里面有内容，则这次写入会写在文件的最末尾。
  • O_CREAT 表示如果指定文件不存在，则创建这个文件
  • O_EXCL 表示如果要创建的文件已存在，则出错，同时返回 -1，并且修改 errno 的值。
  • O_TRUNC 表示截断，如果文件存在，并且以只写、读写方式打开，则将其长度截断为0。
  • O_NOCTTY 如果路径名指向终端设备，不要把这个设备用作控制终端。
  • O_NONBLOCK 如果路径名指向 FIFO/块文件/字符文件，则把文件的打开和后继 I/O设置为非阻塞模式 （nonblocking mode）
    以下三个常量同样是选用的，它们用于同步输入输出
  • O_DSYNC 等待物理 I/O 结束后再 write。在不影响读取新写入的数据的前提下，不等待文件属性更新。
  • O_RSYNC read 等待所有写入同一区域的写操作完成后再进行
  • O_SYNC 等待物理 I/O 结束后再 write，包括更新文件属性的 I/O
  1. mode：
    mode表示设置文件访问权限的初始值，和用户掩码umask有关，比如0644表示-rw-r–r–也可以选择用S_IRUSR、S_IWUSR等宏定义按位或起来表示。要注意的是，有以下几点：
  • 文件权限由open的mode参数和当前进程的umask掩码共同决定。
  • 第三个参数是在第二个参数中有O_CREAT时才作用，如果没有，则第三个参数可以忽略

返回值:
返回一个新的回文件描述符,若返回-1则表示调用失败

lseek函数

参数:

  1. fd:
    文件描述符
  2. offset:
    偏移量
  3. whence:
  • SEEK_SET设置文件指针偏移量
  • SEEK_CUR设置偏移量：当前位置+offset
  • SEEK_END设置偏移量：文件大小+offset

返回值:
文件指针的位置
作用：

  1. 移动指针到文件头
    iseek(fd,0,SEEK_SET)
  2. 获取当前文件位置
    iseek(fd,0, SEEK_CUR)
  3. 获取文件长度
    iseek(fd,0, SEEK_END)
  4. 拓展文件长度
    iseek(fd,100, SEEK_END)

lread函数

参数:

  1. fd:文件描述符
  2. buf:需要读取数据存放的地方，数组地址
  3. count:数组大小

返回值:

  1. >0:操作成功，返回实际读取到的字节数
  2. =0:文件读取完毕
  3. <0:失败

write函数

参数：

  1. fd:文件描述符
  2. buf:需要写入的数据
  3. count:数据大小

返回值:

  1. >0:返回写入的字节数
  2. =0:未写入任何内容
  3. <0:失败

stat函数

   int fstat(int fd, struct stat *statbuf);
   int lstat(const char *pathname, struct stat *statbuf);
