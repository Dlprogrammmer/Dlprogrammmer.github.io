PWM

发表于2024-01-28|更新于2024-02-16|嵌入式
|阅读量:

概念

PWM（Pulse Width Modulation，脉宽调制）是一种通过改变信号的脉冲宽度来控制电平的技术。在实际应用中，PWM常用于控制各种设备，如电机的速度调节、灯光的亮度调节以及音频的音量调节等。在STM32微控制器中，我们可以利用其内置的PWM外设来生成PWM信号，实现对外部设备的精确控制。

原理

PWM信号由高电平和低电平组成。通过改变高电平和低电平的时间比例，可以实现对输出电平的控制。调整高电平时间比例可以改变输出电平的平均值，从而实现对设备的控制。

PWM模式

PWM模式1
向上计数：CNT< CCR时，REF置有效电平，CNT≥CCR时，REF置无效电平
向下计数：CNT> CCR时，REF置无效电平，CNT≤CCR时，REF置有效电平
PWM模式2
向上计数：CNT< CCR时，REF置无效电平，CNT≥CCR时，REF置有效电平
向下计数：CNT> CCR时，REF置有效电平，CNT≤CCR时，REF置无效电平

参数

  • 周期：周期是指PWM信号一个完整的周期所需要的时间。它包括一个高电平部分和一个低电平部分。周期由时钟源或计数器决定，并以时间单位（例如微秒、毫秒）表示。
  • 频率：频率是周期的倒数，表示PWM信号的重复率。以赫兹（Hz）为单位，即每秒钟发生的周期次数，常见的PWM频率范围从几十赫兹至数千赫兹不等。
  • 占空比：占空比是PWM信号的高电平时间占整个周期的百分比。通常用百分比表示，可以根据需求调整。占空比为50%时，信号处于平均电压位置，称为50%的占空比点。
  • 脉冲宽度：脉冲宽度是指PWM信号高电平的持续时间。脉冲宽度直接决定了PWM信号的实际效果，例如在LED调光中，脉冲宽度越长，LED亮度越高。
  • 分辨率：分辨率是指PWM信号幅度的细化程度。它表示将周期划分为多少个离散级别，决定了PWM信号的精度。较高的分辨率可以提供更平滑的输出，但可能需要更高的计算资源。

参数计算

PWM频率： Freq = CK_PSC / (PSC + 1) / (ARR + 1)
PWM占空比： Duty = CCR / (ARR + 1)
PWM分辨率： Reso = 1 / (ARR + 1)

PWM控制舵机

舵机是一种根据输入PWM信号占空比来控制输出角度的装置，输入PWM信号要求：周期为20ms，高电平宽度为0.5ms~2.5ms，500-2500us的PWM高电平部分对应控制180度舵机的0-180度。
以180度角度伺服为例，那么对应的控制关系是这样的：
0.5ms————–0度；
1.0ms————45度；
1.5ms————90度；
2.0ms———–135度；
2.5ms———–180度；

PWM驱动电机

直流电机是一种将电能转换为机械能的装置，有两个电极，当电极正接时，电机正转，当电极反接时，电机反转。直流电机属于大功率器件，GPIO口无法直接驱动，需要配合电机驱动电路来操作。TB6612是一款双路H桥型的直流电机驱动芯片，可以驱动两个直流电机并且控制其转速和方向。在电机控制中，电压越大，电机转速越快，而通过PWM输出不同的模拟电压，便可以使电机达到不同的输出转速。
